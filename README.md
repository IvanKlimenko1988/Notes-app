# Приложение "Заметки"
## Разработчик: Клименко Иван
### Информация о проекте:

Проект содержит функционал работы с заметками.

Функции:
+ создавать заметку
+ сохранять заметку
+ читать список заметок
+ прочитать заявку по номеру
+ редактировать заметку
+ удалять заметку
+ загружать заметки из файла
